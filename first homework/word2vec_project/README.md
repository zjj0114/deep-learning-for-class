# Word2Vec 词向量训练项目

## 项目说明

本项目使用Word2Vec算法训练词向量，并可视化展示词向量的分布。通过此项目，你可以了解：

- Word2Vec模型的基本原理
- 如何使用gensim库训练词向量
- 如何可视化词向量的分布

## 项目结构

```
word2vec_project/
├── data/             # 存放训练数据
│   └── requirements.txt  # 示例训练文本
├── models/           # 存放训练好的模型
│   └── word2vec.model  # 训练好的Word2Vec模型
├── main.py           # 主代码文件
└── README.md         # 项目说明
```

## 环境要求

- Python 3.11+


## 使用方法

1. 克隆项目到本地
2. 运行主代码：`python main.py`
3. 查看生成的词向量分布图像，在output文件夹里

## 数据集说明

本项目使用了一个简单的示例文本文件`requirements.txt`，包含了一些常见的英文句子，用于训练Word2Vec模型。

## 模型训练

模型使用gensim库的Word2Vec实现，训练参数如下：

- 向量维度：100
- 窗口大小：5
- 最小词频：1
- 训练轮次：10

## 词向量可视化

使用t-SNE算法将高维词向量降维到2维空间，并使用matplotlib绘制词向量的分布。

## 结果展示

运行代码后，会生成一个包含10个词向量分布的图像，展示词向量之间的相似性和关系。
